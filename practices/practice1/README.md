# Лабораторная работа №3: Динамическая память и OpenMP

## Описание
Программа демонстрирует работу с динамическим массивом, его заполнением случайными числами и параллельным вычислением суммы элементов с использованием OpenMP.

## Требования
- Компилятор C++ с поддержкой OpenMP (например, GCC/MinGW).

## Как запустить

### 1. Компиляция
Откройте терминал в папке `practice1` и выполните команду:

```bash
g++ -fopenmp lab3.cpp -o lab3
```

> **Примечание:** Флаг `-fopenmp` обязателен для включения поддержки параллельных вычислений.

### 2. Запуск
После успешной компиляции выполните:

**Windows (PowerShell/CMD):**
```powershell
.\lab3.exe
```

**Linux/macOS:**
```bash
./lab3
```

## Использование
1. При запуске программа попросит ввести размер массива.
2. Введите целое положительное число (например, `1000000`).
3. Программа выведет:
   - Сообщение о заполнении массива.
   - Сумму элементов (вычисленную параллельно).
   - Среднее значение.
   - Сообщение об освобождении памяти.
